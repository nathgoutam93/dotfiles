## YAZI
using default yazi installation and [mikavilpas/yazi.nvim](https://github.com/mikavilpas/yazi.nvim) plugin to work with neovim.
except for image previews i'm using [viu](https://github.com/atanunq/viu) other than that recommended CLI tools are installed. 

## NEOVIM 

- [Lazy](https://github.com/folke/lazy.nvim) as plugin manager  
- [Mason](https://github.com/mason-org/mason.nvim) to install needed LSP, Formatters, Linters

### THEME
- [onedark-dark](https://github.com/olimorris/onedarkpro.nvim) colorscheme
- using [nhattVim/alpha-ascii.nvim](https://github.com/nhattVim/alpha-ascii.nvim) to toggle and load ascii arts generated using [img2art](https://github.com/Asthestarsfalll/img2art)

## Bootstrap
- Simply run the `bootstrap.sh` for linux/mac systems or `bootstrap.ps1` for windows machine on powershellv5.0 or greater.
- most things will work correctly for unix like systems but for windows some thinks might not work from get go. (eg: image previewer and nvim as yazi default opener)
